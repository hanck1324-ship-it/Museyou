# 네비게이션 - Wireframe

## 개요
사용자가 앱 내에서 주요 섹션 간 이동을 할 수 있도록 하는 네비게이션 컴포넌트의 와이어프레임입니다.

## 레이아웃 구조

```
┌─────────────────────────────────────┐
│         Navigation Bar              │
│  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐  │
│  │Icon │ │Icon │ │Icon │ │Icon │  │
│  │Label│ │Label│ │Label│ │Label│  │
│  └─────┘ └─────┘ └─────┘ └─────┘  │
└─────────────────────────────────────┘
```

## 네비게이션 구성 요소

### 1. 네비게이션 컨테이너
- **위치**: 하단 고정 (Bottom Navigation) 또는 상단 고정
- **너비**: 전체 화면 너비 (100%)
- **높이**: 64px (모바일), 72px (태블릿 이상)
- **배경**: 반투명 또는 단색
- **정렬**: 수평 정렬, 아이템 균등 분배

### 2. 네비게이션 아이템 (개별)
각 네비게이션 아이템은 다음으로 구성됩니다:

```
┌───────────────┐
│   [아이콘]     │  24x24px
│   라벨 텍스트   │  12px
└───────────────┘
```

- **아이콘 영역**: 24x24px
- **라벨 영역**: 12px 폰트 크기
- **패딩**: 상하 8px, 좌우 16px
- **최소 터치 영역**: 48x48px

### 3. 활성 상태 표시
- 현재 활성화된 페이지의 아이템은 시각적으로 강조
- 방법: 색상 변경, 하단 인디케이터, 배경 색상 등

## 네비게이션 아이템 목록

### 기본 4개 아이템 구성
1. **홈**
   - 아이콘: House/Home
   - 라벨: "홈"
   - 링크: `/`

2. **공연**
   - 아이콘: Music/Theater
   - 라벨: "공연"
   - 링크: `/performances`

3. **매칭**
   - 아이콘: Heart/Users
   - 라벨: "매칭"
   - 링크: `/matching`

4. **마이페이지**
   - 아이콘: User/Profile
   - 라벨: "마이"
   - 링크: `/profile`

## 반응형 고려사항

### 모바일 (< 768px)
- 하단 고정 네비게이션
- 4개 아이템까지 표시
- 아이콘 + 라벨 조합

### 태블릿/데스크톱 (≥ 768px)
- 좌측 사이드바 또는 상단 네비게이션
- 더 많은 아이템 표시 가능
- 확장된 라벨 표시

## 인터랙션

### 상태
1. **Default (기본)**: 비활성 상태
2. **Hover (호버)**: 마우스 오버 시
3. **Active (활성)**: 현재 페이지
4. **Pressed (눌림)**: 클릭/탭 시

### 애니메이션
- 페이지 전환 시 부드러운 색상 전환 (200ms)
- 아이콘 크기 변화 (선택사항)
- 활성 인디케이터 슬라이드 효과

## 접근성
- ARIA 라벨 적용
- 키보드 네비게이션 지원
- 스크린 리더 호환
- 최소 터치 영역 준수 (48x48px)
