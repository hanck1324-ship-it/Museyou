# 네비게이션 - UI 디자인

## 디자인 시스템

### 컬러 팔레트

#### 기본 상태
- **배경색**: `bg-white` / `bg-gray-900` (다크모드)
- **아이콘 색상**: `text-gray-500`
- **라벨 색상**: `text-gray-600`
- **보더**: `border-t border-gray-200`

#### 활성 상태
- **아이콘 색상**: `text-primary-600`
- **라벨 색상**: `text-primary-600`
- **인디케이터**: `bg-primary-600`

#### 호버 상태
- **배경색**: `bg-gray-50`
- **아이콘 색상**: `text-gray-700`

### 타이포그래피

- **라벨 폰트**: 
  - Font Family: `font-sans`
  - Font Size: `text-xs` (12px)
  - Font Weight: `font-medium` (500)
  - Line Height: `leading-tight`

### 간격 및 크기

```css
/* 네비게이션 컨테이너 */
height: 64px (mobile), 72px (tablet+)
padding: 0 safe-area-inset-bottom
background: backdrop-blur-lg

/* 네비게이션 아이템 */
padding: 8px 16px
gap: 4px (아이콘과 라벨 사이)
min-width: 64px

/* 아이콘 */
width: 24px
height: 24px
stroke-width: 2px

/* 활성 인디케이터 (선택사항) */
width: 32px
height: 3px
border-radius: 1.5px
```

## 다크모드 지원

```css
/* 다크모드 스타일 */
.dark {
  /* 컨테이너 */
  background: rgba(17, 24, 39, 0.9); /* bg-gray-900/90 */
  border-color: rgba(55, 65, 81, 1); /* border-gray-700 */
  
  /* 비활성 아이템 */
  color: rgba(156, 163, 175, 1); /* text-gray-400 */
  
  /* 호버 */
  hover-background: rgba(31, 41, 55, 1); /* bg-gray-800 */
  
  /* 활성 */
  active-color: rgba(99, 102, 241, 1); /* text-primary-500 */
}
```

## 반응형 디자인

### 모바일 (< 768px)
```css
.navigation-mobile {
  position: fixed;
  bottom: 0;
  height: 64px;
  flex-direction: row;
}

.nav-item-mobile {
  flex-direction: column;
  font-size: 12px;
}
```

### 태블릿/데스크톱 (≥ 768px)
```css
.navigation-desktop {
  position: fixed;
  top: 0; /* 또는 left: 0 for sidebar */
  height: 72px; /* 또는 100vh for sidebar */
  flex-direction: row; /* 또는 column for sidebar */
}

.nav-item-desktop {
  padding: 12px 24px;
  font-size: 14px;
}
```

## 애니메이션 & 효과

### 전환 효과
```css
.nav-item {
  transition: all 200ms cubic-bezier(0.4, 0, 0.2, 1);
}

.nav-icon {
  transition: transform 150ms ease-out;
}

.nav-item:active .nav-icon {
  transform: scale(0.95);
}
```

### 활성 인디케이터 애니메이션
```css
.active-indicator {
  animation: slideIn 300ms ease-out;
}

@keyframes slideIn {
  from {
    opacity: 0;
    transform: translateX(-50%) scaleX(0);
  }
  to {
    opacity: 1;
    transform: translateX(-50%) scaleX(1);
  }
}
```
