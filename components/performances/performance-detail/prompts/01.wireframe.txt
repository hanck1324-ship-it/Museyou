# Performance Detail - 와이어프레임

## 개요
공연 상세 정보를 보여주는 모달 Dialog 컴포넌트입니다.
Tab 기반으로 정보를 구조화하여 표시합니다.

## 레이아웃 구조

```
┌─────────────────────────────────────────┐
│  [X]  공연 제목                          │  <- Dialog Header
└─────────────────────────────────────────┘
┌─────────────────────────────────────────┐
│                                          │
│         [공연 메인 이미지]                │  <- 이미지 영역 (256px)
│                                          │
└─────────────────────────────────────────┘
┌─────────────────────────────────────────┐
│  [카테고리] [지역]        ⭐ 4.5 (123)   │  <- 메타 정보
└─────────────────────────────────────────┘
┌─────────────────────────────────────────┐
│  [공연정보]  [장소/경로]  [리뷰]         │  <- Tabs
└─────────────────────────────────────────┘
┌─────────────────────────────────────────┐
│                                          │
│          탭별 콘텐츠 영역                 │
│                                          │
│  공연정보: 설명, 주최, 관람료, 일시       │
│  장소/경로: 지도, 교통편, 길찾기          │
│  리뷰: 사용자 리뷰 목록                   │
│                                          │
└─────────────────────────────────────────┘
```

## 구성 요소

### 1. Dialog Container
- 최대 너비: 2xl (672px)
- 최대 높이: 90vh (스크롤 가능)
- 반응형: 모바일 95vw, 데스크톱 full width
- 배경: 반투명 overlay
- 닫기: ESC 키 또는 X 버튼

### 2. Header
- 공연 제목 (최대 2줄)
- 닫기 버튼 (우측 상단)
- 패딩: 우측 여백 24px

### 3. 메인 이미지
- 크기: 높이 192px (모바일), 256px (데스크톱)
- 비율: 16:9
- 둥글기: rounded-lg
- 오버플로우: hidden

### 4. 메타 정보
- Badges: 카테고리, 지역
- 평점: 별 아이콘 + 숫자 + 리뷰 수
- 레이아웃: flex space-between

### 5. Tabs
#### Tab List
- 3개 탭: 공연정보, 장소/경로, 리뷰
- Grid layout (3열 균등)
- 높이: auto
- 텍스트: 모바일 12px, 데스크톱 14px

#### Tab Content
- 패딩 상단: 16px
- 공백: 12-16px

## Tab 3: 리뷰

```
┌─────────────────────────────────┐
│  [ReviewSection 컴포넌트]        │
│  - 리뷰 작성 폼                   │
│  - 리뷰 목록                      │
│  - 무한 스크롤                    │
└─────────────────────────────────┘
```

## 반응형

### 모바일 (< 640px)
- Dialog width: 95vw
- 이미지 높이: 192px
- 탭 텍스트: 12px
- 탭 높이: auto (padding 8px)
- 그리드: 1열로 변경 가능

### 태블릿 (640px - 1024px)
- Dialog width: full (최대 672px)
- 이미지 높이: 224px

### 데스크톱 (> 1024px)
- Dialog width: full (최대 672px)
- 이미지 높이: 256px
- 탭 텍스트: 14px

## 상태

### 열림/닫힘
- open prop으로 제어
- onOpenChange 콜백으로 상태 전달
- 배경 클릭 또는 ESC로 닫기

### 탭 전환
- defaultValue: "info"
- 클릭 시 탭 전환
- URL 파라미터 동기화 (선택)

## 애니메이션
- Dialog 열림: fade + scale
- 탭 전환: fade
- 이미지 로드: skeleton

## 접근성
- Dialog role
- 탭 키보드 네비게이션
- ESC 키로 닫기
- Focus trap (Dialog 내부)
- ARIA 라벨
