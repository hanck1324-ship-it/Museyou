# Performance Card - 와이어프레임

## 개요
공연 정보를 카드 형태로 보여주는 컴포넌트입니다.
목록 페이지에서 여러 개의 카드가 그리드 형태로 표시됩니다.

## 레이아웃 구조

```
┌─────────────────────────────────┐
│   [공연 이미지]                   │  <- 이미지 영역 (180px 높이)
│   category badge (우측 상단)      │
│   hover: 확대 효과                │
└─────────────────────────────────┘
┌─────────────────────────────────┐
│  [공연 제목]          ⭐ [평점]  │  <- 헤더 영역
└─────────────────────────────────┘
┌─────────────────────────────────┐
│  📍 [장소]                       │  <- 정보 영역
│  📅 [날짜]                       │
│  🕐 [시간]                       │
│  [지역] badge     [가격]         │
└─────────────────────────────────┘
┌─────────────────────────────────┐
│  [상세정보 버튼]    [하트 아이콘] │  <- 액션 영역
└─────────────────────────────────┘
```

## 구성 요소

### 1. 이미지 영역
- 크기: 180px (모바일), 192px (데스크톱)
- 비율: 16:9
- Fallback 이미지 처리
- Hover 시 1.1배 확대
- 상단 그라데이션 오버레이 (hover 시)

### 2. 카테고리 Badge
- 위치: 이미지 우측 상단 (absolute)
- 스타일: 흰색 배경, 반투명
- 텍스트: 클래식/연극/뮤지컬/전시 등

### 3. 헤더 영역
- 제목: 최대 2줄 (line-clamp-2)
- 평점: 별 아이콘 + 숫자
- 제목 hover 시 색상 변경 (emerald)

### 4. 정보 영역
- 아이콘 + 텍스트 조합
- 장소, 날짜, 시간 정보
- 지역 badge + 가격 (양쪽 정렬)

### 5. 액션 영역
- 상세정보 버튼 (flex-1)
- 하트 아이콘 버튼 (optional)
- 그라데이션 배경 (emerald → green)

## 반응형

### 모바일 (< 640px)
- 카드 1열
- 패딩: 12px
- 폰트: 작게 (text-xs, text-sm)
- 버튼 높이: 32px
- 아이콘: 12px

### 태블릿 (640px - 1024px)
- 카드 2열
- 패딩: 16px

### 데스크톱 (> 1024px)
- 카드 3-4열
- 패딩: 24px
- 폰트: 기본 크기
- 버튼 높이: 40px
- 아이콘: 16px

## 상태

### Default
- 배경: white/90 (반투명)
- 보더: 2px solid (기본 색상)
- 그림자: 기본

### Hover
- Y축 이동: -4px
- 보더: emerald-200
- 그림자: 2xl
- 이미지: 1.1배 확대
- 제목: emerald-600

### 클릭 (Active)
- 스케일: 0.98

## 애니메이션
- 모든 전환: 300ms ease
- 이미지 확대: 500ms
- 그라데이션 페이드: 300ms

## 접근성
- 카드 전체를 button 역할로 처리 가능
- 키보드 네비게이션 지원
- ARIA 라벨 추가
- 대비율 준수 (WCAG AA)
