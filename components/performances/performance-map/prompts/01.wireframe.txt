# Performance Map - 와이어프레임

## 컴포넌트 개요
공연장 위치를 표시하는 Kakao Map 컴포넌트

## 레이아웃 구조

```
PerformanceMap
├── Container (relative)
│   ├── Loading Overlay (absolute, z-10)
│   │   ├── Loader Icon (animate-spin)
│   │   └── Loading Text
│   │
│   └── Map Container (h-40 sm:h-48)
│       └── Kakao Map (div ref)
```

## 반응형 디자인

### 모바일 (< 640px)
- 높이: 160px (h-40)
- 최소 높이: 160px

### 데스크톱 (≥ 640px)
- 높이: 192px (h-48)

## 상태 정의

### 1. 초기 상태
- `isLoading: true`
- Map SDK 로드 대기
- 로딩 오버레이 표시

### 2. 로딩 완료
- `isLoading: false`
- Map 표시
- 마커 및 인포윈도우 표시

### 3. 에러 상태
- `error: string | null`
- 에러 메시지 표시
- Fallback UI (MapPin 아이콘 + 메시지)

### 4. Venue 없음
- `venue: null | undefined`
- Fallback UI 표시

## 인터랙션

### Map 초기화
1. `useEffect`에서 `venue` 변경 감지
2. Kakao Map SDK 로드 확인
3. Map 생성 (center: venue.lat/lng, level: 3)
4. 마커 생성 및 표시
5. 인포윈도우 생성 및 표시

### 지도 옵션
- 확대 레벨: 3 (1-14, 작을수록 확대)
- 중심점: venue 좌표 또는 기본값 (서울 시청)

## Fallback UI

### Venue 없을 때
```
┌─────────────────────────┐
│                         │
│      [MapPin Icon]      │
│                         │
│   장소 정보가 없습니다   │
│                         │
└─────────────────────────┘
```

### 에러 발생 시
```
┌─────────────────────────┐
│                         │
│      [MapPin Icon]      │
│                         │
│    [에러 메시지]         │
│                         │
└─────────────────────────┘
```

## 마커 및 인포윈도우

### 마커
- 위치: venue 좌표
- 기본 마커 스타일 사용

### 인포윈도우
- 내용:
  - 제목: venue.name
  - 주소: venue.address
- 스타일:
  - 배경: white
  - 둥글기: 8px
  - 그림자: 0 2px 8px rgba(0,0,0,0.1)
