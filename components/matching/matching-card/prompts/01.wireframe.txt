# Matching Card - 와이어프레임

## 개요
문화 데이트 매칭 후보자를 카드 형태로 보여주는 컴포넌트입니다.
스와이프 또는 버튼 클릭으로 상호작용할 수 있습니다.

## 레이아웃 구조

```
┌──────────────────────────────────┐
│                                   │
│     [사용자 프로필 이미지]         │  <- 이미지 영역 (224-256px)
│                                   │
│  [매칭률 Badge] (우측 상단)       │  <- 매칭 점수
│                                   │
│  ┌────────────────────────────┐  │
│  │ [이름, 나이]                │  │  <- 하단 오버레이
│  │ 📍 [지역]                   │  │
│  └────────────────────────────┘  │
└──────────────────────────────────┘
┌──────────────────────────────────┐
│  [자기소개]                       │  <- 콘텐츠 영역
│                                   │
│  [관심사 Badges ...]              │
│                                   │
│  ┌────────────────────────────┐  │
│  │ 📅 함께 보고 싶어 해요      │  │  <- 공연 정보 박스
│  │    [공연 제목]              │  │
│  │    [날짜]                   │  │
│  └────────────────────────────┘  │
└──────────────────────────────────┘
┌──────────────────────────────────┐
│  [프로필 보기]  💬  ❤️          │  <- 액션 버튼들
└──────────────────────────────────┘
```

## 구성 요소

### 1. 이미지 영역
- 크기: 224px (모바일), 256px (데스크톱)
- 비율: 3:4 (세로 비율)
- 그라데이션 오버레이 (하단)
- Hover 시 1.1배 확대
- 성별에 따라 보더 색상 변경
  - 남성: emerald
  - 여성: pink

### 2. 매칭률 Badge
- 위치: 우측 상단 (absolute)
- 아이콘: ✨ (반짝이, animate-pulse)
- 배경: 그라데이션
  - 남성: emerald → teal
  - 여성: pink → rose
- 텍스트: 매칭률 % 표시

### 3. 하단 오버레이
- 배경: black/90 그라데이션
- 텍스트 색상: white
- 이름, 나이
- 위치 아이콘 + 지역

### 4. 콘텐츠 영역
- 자기소개 (2줄 제한)
- 관심사 Badges (최대 3개 표시)
  - 성별별 색상
  - 남성: emerald-100/700
  - 여성: pink-100/700
- 나머지 개수 표시 (+N)

### 5. 공연 정보 박스
- 배경: 연한 색상 (성별별)
- 보더: 미세한 보더
- 아이콘: 📅
- 공연 제목 (1줄 제한)
- 날짜 정보

### 6. 액션 버튼
- 프로필 보기 (outline, flex-1)
- 메시지 보내기 (outline, icon)
- 좋아요 (gradient, icon)

## 반응형

### 모바일 (< 640px)
- 이미지 높이: 224px
- 패딩: 12px
- 폰트: text-xs, text-sm
- 버튼 높이: 32px
- 아이콘: 12px

### 데스크톱 (> 640px)
- 이미지 높이: 256px
- 패딩: 24px
- 폰트: text-sm, text-base
- 버튼 높이: 40px
- 아이콘: 16px

## 상태

### Default
- 배경: white/90
- 보더: 2px (성별별 색상)
- 그림자: 기본

### Hover
- Y축 이동: -4px
- 보더 색상 강조
- 그림자: 2xl
- 이미지 확대: 1.1배

### Like 애니메이션
- 하트 확대 + 회전
- 파티클 이펙트 (선택)
- 카드 슬라이드 아웃 (오른쪽)

### Dislike 애니메이션
- 카드 슬라이드 아웃 (왼쪽)

## 인터랙션

### 스와이프 (모바일)
- 왼쪽 스와이프: Pass
- 오른쪽 스와이프: Like
- 스와이프 시 카드 회전 효과

### 버튼 클릭
- 프로필 보기: 상세 모달 열기
- 메시지: 채팅 시작
- 하트: Like 액션

## 애니메이션
- 전환: 300ms ease
- 이미지 확대: 500ms
- 스와이프: spring animation
- 좋아요: scale + rotate

## 접근성
- 버튼 역할 명확화
- ARIA 라벨
- 키보드 네비게이션
- 스크린 리더 지원
