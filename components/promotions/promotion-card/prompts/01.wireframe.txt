# Promotion Card - 와이어프레임

## 컴포넌트 개요
지자체 프로모션/문화 프로그램을 표시하는 카드 컴포넌트

## 레이아웃 구조

```
PromotionCard
├── Card Container
│   ├── Image Section (h-36 sm:h-40)
│   │   ├── Image (object-cover, hover:scale-110)
│   │   ├── Gradient Overlay (hover 시 표시)
│   │   └── District Badge (absolute top-left)
│   │
│   ├── Header Section
│   │   ├── Title (line-clamp-2)
│   │   └── Category Badge
│   │
│   └── Content Section
│       ├── Description (line-clamp-2)
│       ├── Organizer (MapPin 아이콘)
│       ├── Date (Calendar 아이콘)
│       └── Link Button (optional, ExternalLink 아이콘)
```

## 반응형 디자인

### 모바일 (< 640px)
- 이미지 높이: 36 (h-36)
- 패딩: 12px (p-3)
- 텍스트 크기: text-xs, text-sm
- 아이콘 크기: size-3, size-4

### 데스크톱 (≥ 640px)
- 이미지 높이: 40 (h-40)
- 패딩: 24px (p-6)
- 텍스트 크기: text-sm, text-lg
- 아이콘 크기: size-4

## 상태 정의

### 기본 상태
- 카드: 정적 배치
- 이미지: 기본 크기
- 오버레이: opacity-0

### Hover 상태
- 카드: translate-y-1 (위로 이동), shadow-2xl
- 이미지: scale-110 (확대)
- 오버레이: opacity-100
- 제목: 텍스트 색상 변경 (purple-600)

### 로딩 상태
- ImageWithFallback 컴포넌트로 처리
- Fallback 이미지 표시

## 인터랙션

- 카드 전체: Hover 시 상승 애니메이션
- 이미지: 부드러운 확대 애니메이션 (duration-500)
- 링크 버튼: 외부 링크로 새 창 열기 (target="_blank")
