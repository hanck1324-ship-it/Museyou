# Authentication Module

## 개요
사용자 인증 및 회원가입 기능을 담당하는 모듈입니다.

## 주요 기능
- 로그인/로그아웃
- 회원가입 (일반 회원 / 예술종사자)
- 휴대폰 본인 인증
- 예술종사자 인증 (기관 이메일 또는 증빙서류)

## 파일 구조
```
components/auth/
  └── AuthDialog.tsx        # 인증 다이얼로그 컴포넌트
```

## 상태 관리
- `isLoggedIn`: 로그인 상태
- `currentUser`: 현재 사용자 정보
- `isArtist`: 예술종사자 여부

## API 엔드포인트
- `POST /auth/signup` - 회원가입
- `POST /auth/login` - 로그인
- `POST /auth/signout` - 로그아웃
- `GET /auth/session` - 세션 확인
- `GET /auth/profile` - 프로필 조회

## 일반 회원 vs 예술종사자

### 일반 회원
- 필수: 이메일, 비밀번호, 이름, 나이, 성별, 휴대폰 인증
- 선택: 지역, 관심 장르, 선호하는 만남 스타일, 자기소개

### 예술종사자
- 필수: 이메일, 비밀번호, 이름, 휴대폰 인증, 예술 분야, 소속 기관
- 인증 방법: 기관 이메일 인증 OR 증빙서류 업로드
- 나이, 성별, 만남 스타일 입력 **제외**

## 프롬프트 예시

### 로그인 기능 수정
```
로그인 페이지에 소셜 로그인 버튼을 추가해주세요.
- 카카오, 구글, 네이버 로그인 지원
- AuthDialog.tsx 수정
```

### 회원가입 필드 추가
```
회원가입 폼에 프로필 사진 업로드 기능을 추가해주세요.
- 이미지 파일만 허용 (jpg, png, webp)
- 최대 5MB 제한
- 미리보기 기능 포함
```

### 인증 방식 변경
```
휴대폰 인증을 실제 SMS API로 연동해주세요.
- handleSendVerificationCode 함수에 실제 API 호출 추가
- handleVerifyCode 함수에 인증번호 확인 로직 추가
```
